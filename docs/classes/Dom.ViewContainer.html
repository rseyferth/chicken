Dom.ViewContainer Class

Extends Core.Object
Defined in: /home/ruben/Projects/Javascript/chicken/src/Dom/ViewContainer.js:9
Module: Dom

A ViewContainer is en DomElement in your application that can contain a rendered View. Each ViewContainer needs a unique name, and the main ViewContainer for the application is always called 'main'.
